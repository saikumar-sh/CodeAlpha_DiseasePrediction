# Disease Prediction from Medical Data

## Objective
Predict disease occurrence using machine learning.

## Dataset
Breast Cancer Dataset (Scikit-Learn)

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Model
Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

## Result
Achieved high prediction accuracy on the test dataset.
